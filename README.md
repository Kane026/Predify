# Predify

Predify is een simpele muziek applicatie gebouwd in C#.

## Features

- Nummers afspelen, stoppen, pauzeren en skippen
- Playlist aanmaken
- Nummers toevoegen aan een playlist
- Playlist inkijken
- Nummers verwijderen uit een playlist
- Albums bekijken

## Installatie

1. Clone de repository
2. Open het project in Visual Studio
3. Run het project met `dotnet run`

## Gebruik

Start het programma en kies een optie uit het menu.
